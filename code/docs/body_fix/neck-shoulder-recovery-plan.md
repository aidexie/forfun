# 右侧颈肩康复计划

## 症状总结

- **位置**：右侧颈肩交界处（肩胛提肌/上斜方肌区域）
- **感觉**：有根筋拉扯着，酸痛
- **缓解动作**：头向右后方歪、手臂伸直
- **诱因**：长时间使用键盘鼠标（程序员 + DOTA2/CS2）
- **规律**：工作日用电脑后复发，周末户外活动时好转
- **持续时间**：约半年
- **严重程度**：早上刷牙都感觉肩部累

## 诊断分析

**可能原因**：肩膀前伸（圆肩）导致肩胛提肌持续紧张

**好消息**：
- 周末户外活动时无症状 → 不是结构性损伤
- 身体有自愈能力，只是恢复速度赶不上损伤累积

## 坐姿核心要点

### 最关键：肩膀不要前伸

```
正确：                    错误：

    头                         头
    |                          |
   肩膀                       肩膀 ←← 前伸
    |                        ↗
   身体                    身体
```

### 完整坐姿检查清单

| 部位 | 正确姿势 |
|------|----------|
| 脚 | 平放地面，大腿与地面平行 |
| 臀部 | 坐满椅子，顶到椅背 |
| 腰部 | 有支撑，保持自然前凸 |
| 背部 | 贴靠椅背 |
| **肩膀** | **自然下沉，不耸肩，不前伸（最重要）** |
| 手肘 | 弯曲90-110度，贴近身体两侧 |
| 手腕 | 中立位，与前臂成直线 |
| 头 | 居中，耳朵在肩膀正上方 |

### 键鼠位置

- 鼠标**紧贴键盘右侧**，不要有间隙
- 键盘放在身体正前方
- 显示器距离50-70cm，顶部与眼睛平齐

## 标准8周康复计划

### 第1-2周：松解与放松期

**目标**：让过度紧张的肌肉放松，恢复基础血液循环

#### 每日必做（早晚各一次，每次10分钟）

1. **热敷**（5分钟）
   - 热毛巾或热水袋敷在右侧颈肩交界处

2. **网球/筋膜球松解**（3分钟）
   - 背靠墙，网球放在右侧肩胛骨内侧缘
   - 找到酸痛点，停留20-30秒，深呼吸

3. **拉伸**（每个动作30秒 × 2次）

   **上斜方肌拉伸**：
   - 右手压在屁股下面
   - 左手轻轻把头拉向左肩

   **肩胛提肌拉伸**：
   - 右手压在屁股下面
   - 头转向左侧45度，低头看左腋窝
   - 左手轻压后脑勺

   **胸肌拉伸**：
   - 右手前臂贴门框，手肘与肩同高
   - 身体向前迈步，感觉胸部拉伸

#### 工作/游戏期间

- 每30分钟起身活动1分钟
- **暂停或大幅减少**高强度鼠标游戏

---

### 第3-4周：轻度激活期

**目标**：唤醒沉睡的稳定肌群

#### 继续第1-2周内容 + 新增练习（每天一次）

1. **肩胛骨时钟运动**（2分钟）
   - 耸肩(12点) → 后夹(3点) → 下沉(6点) → 前放(9点)
   - 顺时针10圈，逆时针10圈

2. **俯卧Y字抬起**（不负重）
   - 趴在床上，双臂成Y字形
   - 轻轻抬起5-10厘米，保持3秒
   - 10次 × 2组

3. **靠墙天使**（Wall Angel）
   - 背靠墙，双臂贴墙成投降姿势
   - 手臂沿墙上下滑动
   - 10次 × 2组

#### 第4周末自我检测

- 刷牙还累吗？
- 酸痛频率减少了吗？
- **如果没有任何改善 → 建议就医**

---

### 第5-6周：基础强化期

**目标**：强化薄弱肌群

#### 强化训练（隔天一次，15-20分钟）

1. **弹力带面拉**（Face Pull）
   - 手肘抬高与肩平，向面部拉
   - 夹紧肩胛骨，停顿2秒
   - 15次 × 3组

2. **弹力带划船**
   - 弹力带绕脚掌，手肘贴身向后拉
   - 12次 × 3组

3. **俯卧T字抬起**
   - 双臂成T字形，轻轻抬起
   - 12次 × 3组

4. **肩胛骨俯卧撑**
   - 俯卧撑姿势，手臂不动
   - 只通过肩胛骨移动身体上下
   - 10次 × 3组

#### 游戏恢复

- 可以开始玩，每局后休息5分钟
- 单日不超过2小时
- 游戏前做2分钟热身

---

### 第7-8周：巩固期

**目标**：建立长期习惯，预防复发

#### 训练升级

- 增加弹力带阻力
- 俯卧Y/T可手持0.5-1kg哑铃

#### 新增：颈部等长收缩

- 手掌放额头，头向前用力，手抵抗，头不动
- 保持5秒，换后脑勺、左侧、右侧
- 每方向5次 × 2组

### 核心改动

| 项目 | 标准版 | 加速版 |
|------|--------|--------|
| 松解频率 | 每天2次 | 每天3-4次 |
| 工作打断 | 每小时1次 | 每25分钟1次 |
| 强化训练开始 | 第5周 | 第2周（轻度） |
| 强化频率 | 隔天 | 每天（降低强度） |

### 加速版日程表

| 时间 | 内容 | 时长 |
|------|------|------|
| 早起 | 热敷 + 拉伸 | 10分钟 |
| 工作中 | 每25分钟起身活动（番茄工作法） | 每次5分钟 |
| 午饭后 | 网球松解 + 拉伸 | 10分钟 |
| 下午3-4点 | 轻度强化（面拉+划船，2组×10次） | 15分钟 |
| 晚上 | 完整强化训练 或 游泳 | 20-30分钟 |
| 睡前 | 拉伸 + 热敷 | 10分钟 |

### 额外加速方法

#### 1. 游泳（强烈推荐）

- 蛙泳和自由泳对肩颈特别好
- 水的浮力减少关节负担
- 每周2-3次，每次30-45分钟

#### 2. 换手用鼠标

- 简单操作（浏览网页、看文档）用左手
- 精确操作时再换回右手
- 分担右侧负担

#### 3. 语音输入

- 写文档、回消息时用语音输入
- 减少打字量

#### 4. 站坐交替

- 升降桌或显示器增高架
- 每小时换一次姿势

### 加速版预期效果

- **1周**：酸痛频率明显减少
- **2-3周**：刷牙不再累，日常活动正常
- **4-5周**：可以正常工作+适度游戏

## 工位调整建议

### 鼠标位置（最重要）

- 紧贴键盘右侧
- 放松肩膀时手能自然够到
- 移动鼠标用前臂带动，不只用手腕

### 显示器

- 顶部与眼睛平齐
- 距离50-70cm
- 正对面部

### 椅子

- 座椅高度：双脚平放地面
- 腰部有支撑
- 背部贴靠椅背

### 键盘选择建议

- 优先考虑：人体工学分体键盘 或 紧凑布局（75%/80%）
- 避免：全尺寸104键（小键盘区会把鼠标推远）

## 警示信号（立即就医）

- 手指麻木、刺痛或无力
- 疼痛放射到手臂
- 夜间痛醒
- 3周后完全没有改善
- 症状明显加重
